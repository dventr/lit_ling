# Abbildungsverzeichnis / Figure Metadata

Diese Abbildungen wurden erstellt mit: **Plotly (Python library) + kaleido export engine**

Alle Abbildungen sind in folgenden Formaten verfügbar:
- **HTML**: Interaktive Version mit Zoom und Hover-Informationen
- **PNG**: 600 dpi Auflösung (für Kombinationsgraphiken gemäß Verlagsvorgaben)
- **PDF**: Vektorformat, skalierbar ohne Qualitätsverlust (bevorzugt für Publikationen)
- **EPS**: Vektorformat (kann aus PDF konvertiert werden falls benötigt)

**Hinweis**: PDF-Dateien sind vollständige Vektorgraphiken und erfüllen alle Verlagsvorgaben. Falls EPS-Format explizit gefordert wird, können die PDF-Dateien mit Adobe Acrobat, Inkscape oder `pdf2ps` konvertiert werden.

## Technische Spezifikationen

- **Schriftart**: Helvetica/Arial (wie in Verlagsvorgaben empfohlen)
- **Schriftgröße**: 8-12 pt in Endgröße
- **Linienstärke**: Mindestens 0,3 pt
- **Farbmodus**: RGB (8 bits per channel)
- **Auflösung PNG**: 600 dpi (Kombinationsgraphiken)
- **Maximale Breite**: 174 mm (großformatige Zeitschriften)
- **Maximale Höhe**: 234 mm

## Farbschema

Das verwendete Farbschema ist:
- **Farbenblindfreundlich**: Optimiert für verschiedene Formen der Farbsehschwäche
- **Hoher Kontrast**: Auch in Schwarzweiß gut erkennbar
- **Konsistent**: Gleiche Farben für gleiche Kategorien über alle Abbildungen hinweg

### Farbzuordnung nach Ideologie:
- Sozial: #AF5F0B (Orange-Braun)
- Grün: #E3DA48 (Gelb-Grün)
- Mittelinks: #830861 (Violett)
- Mitte: #8B0000 (Dunkelrot)
- Mitterechts: #441076 (Dunkelviolett)
- Rechts: #013343 (Dunkelblau)
